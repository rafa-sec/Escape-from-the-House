# 🎮 JavaScript Mini-Games Collection

Welcome to my **JavaScript Mini-Games Repository**!

This project is all about learning, experimenting, and having fun with JavaScript by building small, playable games. Each mini-game here is a hands-on exercise where I explore new concepts, improve my coding skills, and enjoy the creative process.

---

## 🚀 Purpose

The main goal of this repository is simple:

> **Practice JavaScript while having fun building mini-games.**

Instead of just studying theory, I actively apply what I learn by creating interactive projects. This helps me understand concepts more deeply and stay motivated.

---

## 📂 Project Structure

Each mini-game is stored in its own folder:

```
/mini-game-name
  ├── index.html
  ├── style.css
  └── script.js
```

Some projects may include additional assets like images or sounds.


2. Open any game folder

3. Run the `index.html` file in your browser
